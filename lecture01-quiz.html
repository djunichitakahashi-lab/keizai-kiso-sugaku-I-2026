Q1この科目の評価で、中間テストと期末テストの配点は合わせて何点？
A40点
B50点
C80点
D60点
正解: D
中間テスト30点＋期末テスト30点＝60点です。
Q2テストの持ち込みについて正しいのは？
A中間テストは持ち込み可、期末テストは持ち込み不可
B中間・期末ともに持ち込み不可
C中間・期末ともに持ち込み可
D中間テストは持ち込み不可、期末テストは持ち込み可
正解: A
中間テストは持ち込み可（ノート・スライド・電卓OK）。ただし期末テストは持ち込み不可 — 微分の計算は自力でできるようになってほしいからです。
Q320%引きの商品をさらに20%引きにすると、元値から何%引き？
A40%引き
B38%引き
C36%引き
D30%引き
正解: C
0.8×0.8＝0.64 → 1−0.64＝0.36 → 36%引き。割引は「足し算」ではなく「掛け算」です。
Q4経済学で数学を学ぶのはなぜか？ もっとも適切なのは？
A経済学では計算が少ないから、数学は特に重要ではない
B数値に基づく「定量的議論」で、主観でなく客観的に判断するために必要
C経済学の専門用語を暗記するために必要
Dテストで点を取るためだけに必要
正解: B
数字で根拠を示せると、主観を超えて客観的に議論できます。これが「定量的議論」であり、経済学で数学を学ぶ最大の理由です。

Q2f(x)=200xのとき、f(4)の値は？
A400
B600

正解: D

Q3y=200xのグラフの特徴として正しいのは？
A右下がりの直線
B曲線
C原点を通る右上がりの直線
D水平な直線

Q4Δxが表すものは？
Axの合計
Bxの変化量
Cxの平均
Dxの最大値

①確認クイズ（全5問正解でSTEP 4解放）
Q1y=ax+bで、aが表すものは？
Ay軸との交点
B直線の傾き
Cx軸との交点
Dグラフの面積
正解: B
aは傾き＝xが1増えたときのyの変化量を表します。
Q2(1,7)と(3,13)を通る直線の傾きは？
A2
B4
C6
D3
正解: D
a＝(13−7)/(3−1)＝6/2＝3 です。
Q3y=3x+4のグラフで、x=0のときyの値は？
A4
B0
C3
D7
正解: A
x=0を代入するとy=3×0+4=4。これが切片bの値です。
Q4傾きaが負の一次関数のグラフは？
A右上がり
B水平
C右下がり
D垂直
正解: C
a<0なら、xが増えるとyが減る → 右下がりのグラフになります。
Q5一次関数 y = ax + b で、b（切片）が表すものは？
Ax が 1 増えたときの y の増分
B直線の傾き
C直線の長さ
Dx = 0 のときの y の値（y 軸との交点）
正解: D
x = 0 を代入すると y = a×0 + b = b。つまり b は「x = 0 のときの y の値」で、グラフでは y 軸との交点になります。スマホ料金なら「通話 0 分でもかかる基本料金」に相当します。
🔒 5問すべて正解するとSTEP 4が解放されます。

Q1需要関数q=−0.02p+18で、価格p=400のとき需要量qは？
A10
B6
C8
D12
正解: A
q=−0.02×400+18=−8+18=10 です。
Q2需要関数の傾きが負であることの意味は？
A価格が上がると需要が増える
B需要は常に一定
C価格が上がると需要が減る
D需要は価格に関係ない
正解: C
傾き<0 → 価格が上がると需要が減る（逆方向に動く）。これは需要法則です。
Q3q=−0.02p+18で、q=12万冊にするための価格は？
A200円
B400円
C500円
D300円
正解: D
12=−0.02p+18 → 0.02p=6 → p=300 です。
Q4経済学のグラフの慣習で、縦軸に取るのは？
A数量
B価格
C時間
D利益
正解: B
経済学では縦軸＝価格、横軸＝数量が慣習です。
