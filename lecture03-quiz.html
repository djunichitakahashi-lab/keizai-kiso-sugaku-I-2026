Q1二次関数を学ぶメリットとして、この回の説明に最も近いものはどれか？
A一番よい点やちょうどよい量を探せる
B全ての経済問題を暗算できる
C一次関数を使わなくてよくなる
D価格と数量の関係を考えなくてよくなる
正解: A
二次関数は、最大値・最小値、つまり「一番よい点」を探すための道具として使えます。経済では「もっとも利益が出る数量」を探す場面で活躍します。
Q2完全競争市場で、個々の企業の価格の扱いとして正しいものはどれか？
A価格を上げても販売数量は変わらない
B市場で決まった価格を受け入れる
C必ず好きな価格をつけられる
D需要曲線を無視できる
正解: B
完全競争市場では、個々の企業は市場で決まった価格を受け入れる立場（プライステイカー）として考えます。これに対する独占市場が、この回のメインテーマです。
Q3独占企業の価格決定力について、正しい説明はどれか？
A費用をゼロにできる力
Bどんな価格でも必ず売れる力
C価格や数量をある程度選べるが、需要曲線からは逃げられない力
D市場均衡を計算しなくてよい力
正解: C
独占企業は価格や数量を選べますが、高すぎる価格では需要量が減ります。万能ではなく、需要曲線という「逃げられない縛り」のなかで最適点を探します。
Q4「価格を上げると売上が必ず増える」と言えない理由はどれか？
A価格を上げると販売数量が減ることがあるから
B費用が必ずゼロになるから
C需要量が必ず増えるから
D独占企業には需要が存在しないから
正解: A
高く売れば1個あたりの売上は増えますが、売れる数が減るため、売上や利潤は途中から下がることがあります。この「山の形」が二次関数で表現できます。
Q5利潤の定義として正しいものはどれか？
A利潤 = 売上 + 費用
B利潤 = 価格だけ
C利潤 = 費用 − 売上
D利潤 = 売上 − 費用
正解: D
利潤は企業のもうけで、売上から費用を引いたもの。記号では π = R − C。STEP 5 で R と C を二次関数として組み立てます。

①確認クイズ（全 5 問正解で STEP 3 解放）
Q1市場均衡を最もよく説明しているものはどれか？
A企業の利潤が必ず最大になる点
B需要量と供給量が一致し、過不足がない点
C価格が0になる点
D供給量だけが最大になる点
正解: B
市場均衡は、買いたい量と売りたい量が一致する点。式では Qd = Qs、グラフでは需要曲線と供給曲線の交点です。
Q2需要関数 Qd = −2P + 100 と供給関数 Qs = 3P − 20 の均衡価格 P はいくつか？
AP = 20
BP = 24
CP = 52
DP = 120
正解: B
−2P + 100 = 3P − 20 → 120 = 5P → P = 24。連立方程式の典型パターンで、第2回の復習です。
Q3上の問題で、均衡数量 Q はいくつか？
AQ = 24
BQ = 48
CQ = 52
DQ = 100
正解: C
P = 24 を Qd = −2P + 100 に代入 → Q = −48 + 100 = 52。供給関数 Qs = 3 × 24 − 20 = 52 でも同じ結果になります（検算）。
Q4最初に一次関数で市場を説明した理由として最も適切なものはどれか？
A現実を扱いやすい直線モデルに単純化できるから
B現実の市場が必ず完全な直線だから
C計算をしなくてよいから
D価格と数量が無関係になるから
正解: A
一次関数は、現実の複雑さを扱いやすい形に近似するための「最初のモデル」。これだけでは捉えきれない「最大値」を表現するために二次関数が登場します。
Q5独占企業の売上を考えると二次関数が自然に出てくる理由として正しいものはどれか？
A売上は価格だけで決まるから
B費用を必ず0にできるから
C需要曲線を無視できるから
D価格が数量の一次関数で決まり、それに数量をかけるから
正解: D
たとえば P = 100 − 2Q のとき、売上 R = P × Q = (100 − 2Q) Q = 100Q − 2Q² となり二次関数が現れます。「(一次) × (一次) = (二次)」がポイント。
🔒 5 問すべて正解すると STEP 3 が解放されます。
STEP 3 — 二次関数と放物線

二次関数の基本形

二次関数の一般式と、グラフの「放物線」が上下どちらに開くか、係数 a の絶対値で「細い・広い」がどう変わるかを整理する。利潤関数で「下に開く山」がよく出る理由も押さえる。

  • 利潤関数で「下に開く山」がよく出る経済的な理由が分かる
①確認クイズ（全 5 問正解で STEP 4 解放）
Q1x² の説明として正しいものはどれか？
Ax を 2 回足すこと
Bx を 2 で割ること
Cx に x をかけること
Dx の符号を必ず負にすること
正解: C
x² は x に x をかけること。たとえば (−2)² = 4 と、負の数でも 2 乗すると正になるのがポイント。
Q2二次関数の一般式として正しいものはどれか？
Ay = ax + b
By = a / x
Cy = ax² + bx + c（a ≠ 0）
Dy = b + c
正解: C
二次関数は x² を含む関数で、一般式は y = ax² + bx + c（a ≠ 0）。a = 0 だと一次関数に戻ります。
Q3y = −3x² + 2x + 5 の放物線はどちらに開くか？
A上に開く
B下に開く
C直線になる
D判断できない
正解: B
x² の係数 a = −3 < 0 なので、下に開く放物線。下に開くと頂点が「最大値」になり、利潤最大化問題で重要な形になります。
Q4係数 a の絶対値が大きくなると、放物線の開き方はどうなるか？
A必ず直線になる
B必ず下に開く
C頂点が消える
D細くなる
正解: D
a の符号は開く向きを、絶対値の大きさは「開き具合」を表します。|a| が大きいほど放物線は細くなります（少し動くだけで y が大きく変わる）。
Q5経済への接続として、利潤関数で「下に開く形」がよく出る理由に最も近いものはどれか？
A数量を増やすほど価格や費用の影響で利潤が途中から下がることがあるから
B利潤は必ず直線だから
C費用を考えないから
D価格が常に一定だから
正解: A
数量を増やすには売れる価格を下げる必要があり、費用も増えるため、利潤は途中で頭打ち→下落と「山の形」になります。これが「下に開く二次関数」の経済的な意味。

STEP 4 — 平方完成と最大・最小

頂点を見つける道具

頂点形式 y = a(x − p)² + q と平方完成のロジックを確認し、x 軸との交点を求める道具まで整理する。最大・最小を「頂点」として一発で読み取れるようにする回。

// このSTEPのゴール
  • 頂点形式 y = a(x − p)² + q の頂点が (p, q) であると言える
  • 平方完成で「x の係数を半分にする」理由（(x + k)² = x² + 2kx + k²）が分かる
  • 具体例で頂点座標と最大値・最小値を計算できる
  • x 軸との交点を求めることが「y = 0 の二次方程式を解くこと」だと理解している
①確認クイズ（全 5 問正解で STEP 5 解放）
Q1y = a(x − p)² + q の頂点はどれか？
A(a, q)
B(−p, q)
C(p, q)
D(q, p)
正解: C
頂点形式 y = a(x − p)² + q では、(x − p)² = 0、つまり x = p のとき y が最小（または最大）になり、頂点は (p, q) です。
Q2平方完成で x の係数を半分にする理由として最も適切なものはどれか？
A常に答えが0になるから
Bx² の係数を消すため
Cx軸との交点を必ず1つにするため
D(x+k)² を展開すると真ん中の項が 2kx になるから
正解: D
(x + k)² = x² + 2kx + k²。「真ん中の項が 2kx」なので、x の係数を半分にすると、かっこの中の k がそのまま見えます。
Q3y = −x² + 6x + 4 を平方完成すると、頂点と最大値はどうなるか？
A頂点は (3, 13)、最大値は 13
B頂点は (−3, 13)、最大値は 13
C頂点は (3, −13)、最小値は −13
D頂点は (6, 4)、最大値は 4
正解: A
y = −(x² − 6x) + 4 = −{(x − 3)² − 9} + 4 = −(x − 3)² + 13。頂点 (3, 13)、下に開くため最大値は 13。
Q4x 軸との交点を求めたいとき、まず何をするか？
Ay = 0 と置いて二次方程式を解く
B必ずグラフを手描きする
Cx = 0 と置くだけでよい
D係数 a を消す
正解: A
x 軸上では y = 0。したがって、x 軸との交点を求めることは ax² + bx + c = 0 という二次方程式を解くことです（因数分解 or 解の公式）。
Q5y = (x − 1)(x − 5) の x 軸との交点として正しいものはどれか？
A(0, 1) と (0, 5)
B(1, 0) と (5, 0)
C(−1, 0) と (−5, 0)
D(1, 5) だけ
正解: B
y = 0 とすると (x − 1)(x − 5) = 0 → x = 1 または x = 5。x 軸との交点は (1, 0) と (5, 0)。
🔒 5 問すべて正解すると STEP 5 が解放されます。
STEP 5 — 独占市場の利潤最大化

需要関数から利潤の山を作る

需要関数から売上・費用・利潤を立式し、平方完成で「一番もうかる数量と価格」を求める。第3回のクライマックス。

利潤最大化平方完成約10分
🎬解説動画
▶️
【動画】独占市場の利潤最大化
先生と学生の会話形式で解説します
→ 視聴
// このSTEPのゴール
  • 需要関数 P = 100 − 2Q の係数の意味（Q が 1 増えると P がどれだけ下がるか）が読める
  • 売上 R = P × Q から二次関数が出てくる流れを示せる
  • 利潤 π = R − C を二次関数として組み立てられる
  • 平方完成で利潤最大化数量 Q* と最大利潤 π* を求め、価格 P* まで戻せる
①確認クイズ（全 5 問正解で全 STEP クリア）
Q1需要関数 P = 100 − 2Q の読み方として正しいものはどれか？
A数量 Q が 1 増えると価格 P は 2 下がる
B数量 Q が 1 増えると価格 P は 100 上がる
C数量 Q と価格 P は無関係である
D価格 P は常に 2 である
正解: A
P = 100 − 2Q では、Q が 1 増えると P は 2 下がる。「多く売るには価格を下げる必要がある」という需要曲線の典型形。
Q2需要関数 P = 100 − 2Q のとき、売上 R として正しいものはどれか？
AR = 100 − 2Q
BR = 20Q + 200
CR = 100Q + 2Q²
DR = (100 − 2Q) Q
正解: D
売上は価格 × 数量。R = P × Q = (100 − 2Q) Q = 100Q − 2Q² となり、二次関数が現れます。
Q3P = 100 − 2Q、C = 20Q + 200 のとき、利潤関数として正しいものはどれか？
Aπ = 2Q² + 120Q + 200
Bπ = 100 − 22Q − 200
Cπ = −2Q² + 80Q − 200
Dπ = −2Q² + 20Q + 200
正解: C
R = 100Q − 2Q²、C = 20Q + 200 なので π = R − C = (100Q − 2Q²) − (20Q + 200) = −2Q² + 80Q − 200。Q² の係数が負なので「下に開く山」になります。
Q4π(Q) = −2Q² + 80Q − 200 = −2(Q − 20)² + 600 のとき、利潤最大化数量はどれか？
AQ = 10
BQ = 20
CQ = 60
DQ = 600
正解: B
頂点形式 −2(Q − 20)² + 600 から、頂点の Q 座標は 20。Q² の係数が負（下に開く）なので、Q = 20 で利潤が最大になります。
Q5この例の答えの経済的な読み方として正しいものはどれか？
AQ = 20 で売ると利潤が最大になり、そのとき P = 60、最大利潤は 600 である
BQ = 600 で売ると価格が最大になる
CP = 20 にすれば必ず利潤が最大になる
D数量を増やすほど利潤は必ず増え続ける
正解: A
Q = 20 を需要関数 P = 100 − 2Q に戻すと P = 60。頂点形式から最大利潤は 600。20 個より少なくても多くても利潤は下がります。
