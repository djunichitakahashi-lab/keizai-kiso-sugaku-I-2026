Q1指数が現実に出てくる例として 最も適切なのは？
A一度だけ100円増える
B毎年同じ割合で増える金額
C名前を五十音順に並べる
D固定された学籍番号
正解: B
同じ割合で増える変化は、指数（同じ倍率を何回掛けたか）で表しやすい現象です。金利・SNS拡散・感染拡大などはみんなこのタイプ。
Q2$2^5$ の「底」はどれ？
A5
B32
C2
D$2^5$ 全体
正解: C
底は「何を掛けるか」、指数は「何回掛けるか」。$2^5$ では底 = 2、指数 = 5。
Q3$2^5$ の値は？
A32
B10
C25
D7
正解: A
$2^5 = 2 \times 2 \times 2 \times 2 \times 2 = 32$。掛け算を繰り返した結果です。
Q4指数的な増え方で 注意すべき点は？
A必ず一定額ずつ増える
B後半になるほど差が小さくなる
C最初から必ず大きく見える
D最初は小さく見えても後半で急に大きくなる
正解: D
指数的な変化は、はじめは控えめに見えても 後半で差が爆発的に大きくなる のが特徴。金利や感染拡大で「想定外」が起きるのはこのため。
Q5$3^4$ の値は？
A12
B27
C64
D81
正解: D
$3^4 = 3 \times 3 \times 3 \times 3 = 9 \times 9 = 81$。同じ数を 4 回掛けるのが累乗の意味。
Q6新聞紙を 1回折るたびに厚さが2倍になるとする。10回折るとおおよそ何倍の厚さ？
ヒント: $2^{10} = ?$
A約 20 倍
B約 100 倍
C約 1,000 倍
D約 10,000 倍
正解: C
$2^{10} = 1024 \approx 1{,}000$ 倍。同じ動作の繰り返しでも、指数では 後半で急に大きくなる のが特徴。これが指数的な増え方の威力。

①確認クイズ（全6問正解で トピック 3 が解放されます）
Q1$a^m \times a^n$ の指数はどうなる？
A$m+n$
B$m-n$
C$mn$
D$m \div n$
正解: A
同じ底の掛け算では 指数を足す: $a^m \times a^n = a^{m+n}$。例: $2^3 \times 2^2 = 2^{3+2} = 2^5 = 32$。
Q2$(2^3)^2$ はどれ？
A$2^5$
B$2^1$
C$2^3$
D$2^6$
正解: D
累乗の累乗では 指数を掛ける: $(a^m)^n = a^{mn}$。$(2^3)^2 = 2^{3 \times 2} = 2^6 = 64$。
Q3$2^0$ の値は？
A0
B1
C2
D定義できない
正解: B
任意の正の数 $a$ について $a^0 = 1$。指数法則 $a^m \div a^m = a^{m-m} = a^0$ かつ $a^m / a^m = 1$ から自然に導かれます。
Q4（スライド連動）$7^{12} \div 7^{10}$ を計算すると？
A$7^2 = 49$
B$7^{22}$
C$7^{120}$
D0
正解: A
同じ底の割り算は 指数を引く: $a^m \div a^n = a^{m-n}$。$7^{12} \div 7^{10} = 7^{12-10} = 7^2 = 49$。
Q5$2^{-3}$ の値は？
A$-8$
B$-6$
C$1/8$
D$0$
正解: C
負の指数は 逆数 を表す: $a^{-n} = 1/a^n$。$2^{-3} = 1/2^3 = 1/8$。
Q6$\displaystyle \left(\frac{2}{3}\right)^{-2}$ の値は？
A$4/9$
B$-4/9$
C$9/4$
D$-9/4$
正解: C
$\left(\frac{a}{b}\right)^{-n} = \left(\frac{b}{a}\right)^{n}$。$\left(\frac{2}{3}\right)^{-2} = \left(\frac{3}{2}\right)^{2} = \frac{9}{4}$。

  • $4^x = 64$ のように底を整える応用ができる
①確認クイズ（全6問正解で トピック 4 が解放されます）
Q1$2^x = 8$ を解け。
A$x = 1$
B$x = 2$
C$x = 3$
D$x = 4$
正解: C
$8 = 2^3$ なので、両辺を $2^x = 2^3$ と書き直すと、指数の比較から $x = 3$。
Q2指数方程式 $a^x = b$ を解くときの 基本テクニックは？
A両辺を足し算する
B両辺の指数を掛ける
C底をひっくり返す
D両辺を同じ底にそろえて指数を比較する
正解: D
両辺を同じ底に揃える と、指数の比較だけで $x$ が求まります。これが指数方程式を解く基本テクニック。
Q3$3^x = 81$ を解け。
A$x = 3$
B$x = 5$
C$x = 4$
D$x = 27$
正解: C
$81 = 3 \times 3 \times 3 \times 3 = 3^4$ なので、$3^x = 3^4$ から $x = 4$。
Q4（スライド連動）$4^x = 64$ を解け。
A$x = 2$
B$x = 3$
C$x = 4$
D$x = 16$
正解: B
$64 = 4 \times 4 \times 4 = 4^3$ なので $x = 3$。または底を 2 に揃えると $4^x = (2^2)^x = 2^{2x}$、$64 = 2^6$ で $2x = 6 \Rightarrow x = 3$。
Q5$5^{n+1} = 125$ を解け。
A$n = 1$
B$n = 2$
C$n = 3$
D$n = 25$
正解: B
$125 = 5^3$ なので $5^{n+1} = 5^3$ から $n+1 = 3 \Rightarrow n = 2$。指数の比較 で文字を含む式も解けます。
Q6$9^x = 27$ を解け（底をそろえる応用）。
A$x = 1$
B$x = 3$
C$x = 3/2$
D$x = 2/3$
正解: C
$9 = 3^2$、$27 = 3^3$ と 底を 3 に揃える。$9^x = (3^2)^x = 3^{2x}$、$3^{2x} = 3^3$ から $2x = 3 \Rightarrow x = \frac{3}{2}$。

Q1元本とは何か？
A追加で発生する利息
B利息の割合
C最初のお金（元になる金額）
D返済総額だけを指す言葉
正解: C
元本は「利息計算のもとになる最初のお金」。$A = P(1+r)^t$ の $P$ がこれにあたります。
Q2単利の特徴は？
A最初の元本だけに利息がつく
B利息にも利息がつく
C金利が必ず0になる
D指数が必ず不要になる
正解: A
単利は 最初の元本だけに毎期利息がつく: $A = P(1 + rt)$。利息は一次関数で増えます。
Q3複利の公式として適切なのは？
A$A = P + rt$
B$A = Pr + t$
C$A = P(1 + rt)$
D$A = P(1 + r)^t$
正解: D
複利は 毎期 $(1+r)$ を掛ける ので $t$ 期後は $A = P(1+r)^t$。指数関数型の増え方。
Q4（スライド連動）元本100万円、年利6%、複利で 10年後の元利合計は約いくら？
ヒント: $(1.06)^{10} \approx 1.7908$
A約 160 万円
B約 179 万円
C約 200 万円
D約 260 万円
正解: B
$A = 1{,}000{,}000 \times (1.06)^{10} \approx 1{,}000{,}000 \times 1.7908 \approx 1{,}790{,}800 \approx 約179万円$。
Q5元本 50 万円、年利 4%、単利で 3 年後の元利合計は？
A約 52 万円
B約 56 万円
C約 60 万円
D約 200 万円
正解: B
$A = P(1 + rt) = 50 \times (1 + 0.04 \times 3) = 50 \times 1.12 = 56$ 万円。単利は利息が一定（毎年 2 万円 × 3 年 = 6 万円）。
Q6元利合計とは何か？
A最初に預けた金額のみ
B利息のみ
C元本 + 利息の合計
D金利の年率
正解: C
元利合計とは「元本 + 利息」の合計金額。期末に手元にあるお金の総額です。

Q172の法則で、年利6%なら約何年で2倍？
A約 12 年
B約 6 年
C約 24 年
D約 72 年
正解: A
72の法則: 72 ÷ 年利(%) ≒ 倍増年数。$72 \div 6 = 12$ 年で約2倍に。厳密値 $\log 2 / \log 1.06 \approx 11.9$ 年とよく一致。
Q2借りる側にとって 高金利が危険な理由は？
A返済総額が必ず小さくなるから
B元本が一瞬で0になるから
C金利を見なくてよくなるから
D利息負担が重くなり、元本が減りにくくなることがあるから
正解: D
高金利では支払額の中で 利息の割合が大きくなり、元本が減りにくくなります。これがリボ払いで返済が長引く原因。
Q3元本100万円、年利5%の複利で、等比数列の 共通比は？
A0.05
B5
C1.05
D100万円
正解: C
毎年 $(1 + 0.05) = 1.05$ を掛けるので、複利の元利合計は 共通比 1.05 の等比数列。一般項は $100 \times 1.05^n$ 万円。
Q4（スライド連動）借金20万円、年利18%（月利1.5%）、毎月8,000円返済すると 完済月数は約いくつ？
ヒント: 毎月「残高 × 1.015 − 8,000」を残高が0になるまでExcel等でシミュレーション
A約 25 ヶ月（2.1 年）
B約 32 ヶ月（2.7 年）
C約 48 ヶ月（4.0 年）
D約 60 ヶ月（5.0 年）
正解: B
毎月の残高は $B_{n+1} = B_n \times 1.015 - 8{,}000$ と変化し、残高が 0 になるまで 約 32 ヶ月。Excel/Colab で再帰式を組むと確認できます（厳密な式は対数を使う＝経済学基礎数学IIで扱います）。
Q572の法則で、年利 9% なら約何年で 2 倍？
A約 6 年
B約 7 年
C約 8 年
D約 9 年
正解: C
72の法則: 72 ÷ 年利(%) ≒ 倍増年数。$72 \div 9 = 8$ 年で約2倍に。年利が高いほど 2 倍に届く時間が短くなる。
Q6借金 30万円、年利 18%（月利1.5%）。初月の利息はいくら？
A約 1,500 円
B約 3,000 円
C約 4,500 円
D約 54,000 円
正解: C
初月の利息 = 残高 × 月利 = $300{,}000 \times 0.015 = 4{,}500$ 円。毎月返済しても、最初は利息が多くて元本がなかなか減らない（これがリボ払いの怖さ）。
