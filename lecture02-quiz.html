STEP 1 — 連立方程式の解き方

代入法・加減法

2 つの式を同時に満たす (x, y) を求める「連立方程式」を、代入法と加減法の 2 つで解けるようにする。

// このSTEPのゴール
  • 連立方程式の意味（2 式を同時に満たす解）が分かる
  • 代入法で連立方程式を解ける
  • 加減法で連立方程式を解ける（係数をそろえる手順を含む）
  • 連立方程式の解 = 2 直線の交点、というグラフ的意味が分かる
①確認クイズ（全 5 問正解で STEP 2 解放）
Q1連立方程式の「解」とは何か？
Aどちらか片方の式を満たす (x, y) の組
B2 つの式を「同時に」満たす (x, y) の組
Cx も y も正の値となる組
Dx = y となるような組
正解: B
連立方程式の解 = 2 つの式を「同時に」満たす (x, y) の組。グラフでは 2 直線の交点です。
Q2【代入法】 連立方程式 y = 2x + 1, y = −x + 7 を代入法で解け。(x, y) はいくつ？
A(x, y) = (2, 5)
B(x, y) = (1, 3)
C(x, y) = (0, 7)
D(x, y) = (5, 2)
正解: A
代入法: 両式とも「y = ⋯」なので、右辺どうしを等しいと置く。2x + 1 = −x + 7 → 3x = 6 → x = 2。式①に代入: y = 2×2 + 1 = 5。よって (x, y) = (2, 5)。
Q3【加減法・基本】 連立方程式 x + 3y = 10, x − y = −2 を解け。(x, y) はいくつ？
A(x, y) = (2, 4)
B(x, y) = (3, 1)
C(x, y) = (0, 2)
D(x, y) = (1, 3)
正解: D
加減法で式①−式②: (x+3y)−(x−y) = 10−(−2) → 4y = 12 → y = 3。式②に代入: x − 3 = −2 → x = 1。よって (x, y) = (1, 3)。
Q4【加減法・係数そろえ】 連立方程式 2x + y = 5, x + 3y = 5 を解け。(x, y) はいくつ？
A(x, y) = (0, 5)
B(x, y) = (3, −1)
C(x, y) = (2, 1)
D(x, y) = (1, 2)
正解: C
係数がそろっていないので、まず式①を 3 倍: 6x + 3y = 15。式②を引くと 5x = 10 → x = 2。式①に代入して 4 + y = 5 → y = 1。これが「加減法の典型」（係数そろえ → 引く → 代入）。
Q5連立方程式の解は、2 直線をグラフに描いたとき何にあたる？
A必ず原点 (0, 0) を通る点
Bどちらか片方の直線上のすべての点
C2 直線が平行に並ぶ位置
D2 直線が交わる「交点」
正解: D
連立方程式の解 = 2 つの式を同時に満たす点 = グラフ上では 2 直線が交わる「交点」。Q2 で求めた (2, 5) も、グラフでは y = 2x + 1 と y = −x + 7 の交点になります。

// このSTEPのゴール
  • 切片と傾きの意味を説明できる
  • 需要曲線がみぎさがり、供給曲線がみぎあがりになる理由が分かる
  • q = ... の式を p = ... に書き直してグラフを描ける（需要・供給の両方）
  • 需要・供給のシフトの向きと意味が分かる
①確認クイズ（全 5 問正解で STEP 3 解放）
Q1一次関数 y = ax + b の 切片 b の意味として正しいのは？
A直線の長さ
Bx = 0 のときの y の値（y 軸との交点）
Cy = 0 のときの x の値
Dx が 1 進んだとき y が変わる量
正解: B
切片 b は x = 0 のときの y の値、つまり直線が y 軸と交わる位置のスタート地点です。
Q2一次関数 y = ax + b の 傾き a の意味として正しいのは？
A直線が止まる場所
Bx が 1 進んだときに y が変わる量
Cx の最大値
Dx = 0 のときの y の値
正解: B
傾き a は「x を 1 進めたら y がどれだけ上がるか（または下がるか）」。a が正なら右上がり、負なら右下がりになります。
Q3需要曲線と供給曲線の傾きの向きとして正しいのは？
A両方とも右上がり
B両方とも右下がり
C需要曲線は右上がり、供給曲線は右下がり
D需要曲線は右下がり、供給曲線は右上がり
正解: D
需要曲線は「値段↑→買いたい量↓」で右下がり（傾き負）、供給曲線は「値段↑→売りたい量↑」で右上がり（傾き正）。
Q4【供給関数の立式】 供給関数 q = S(p) = 2p をグラフ用に p = ... の形に書き直すと？
Ap = q / 2
Bp = −q / 2
Cp = q + 2
Dp = 2q
正解: A
q = 2p の両辺を 2 で割ると p = q/2。グラフでは原点 (0, 0) を通り、傾き 1/2 の右上がり直線（みぎあがり）になります。
Q5所得が増えて、ある商品の需要が大きく増えた。需要曲線はどう動く？
A左シフト
B傾きが急になる
C右シフト（同じ価格でもたくさん買いたい人が増える）
D動かない
正解: C
所得増・人気上昇のような「価格以外の要因」が需要を増やすと、曲線そのものが右に平行移動します。形は変わらず位置だけ動くのがシフトです。
📝 小課題対応: 第二回 演習問題 4(a), 4(b)（需要・供給関数を p = ⋯ に変形してグラフを描く）

STEP 3 — 市場均衡

需要 = 供給を解く

需要関数と供給関数を連立させて均衡価格 p* と均衡数量 q* を求める。均衡からズレた場合の超過需要・超過供給、シフトによる均衡変化も整理する。

均衡超過需要・供給シフト

先生と学生の会話形式で解説します
→ 視聴
// このSTEPのゴール
  • 市場均衡の意味（需要量 = 供給量）が分かる
  • 連立方程式で均衡価格・均衡数量を求められる
  • 均衡からズレた価格で「超過需要」「超過供給」が起きることを判断できる
  • 需要・供給シフトによる均衡変化（4 パターン）の方向を覚えている
①確認クイズ（全 6 問正解で STEP 4 解放）
Q1市場均衡とはどのような状態か？
A生産者がもっとも儲かる状態
B商品が品切れになっている状態
C価格が最も安くなる状態
D需要量と供給量が一致し、過不足のない状態
正解: D
市場均衡 = 需要量と供給量が一致する状態。グラフでは需要曲線と供給曲線の交点で、過不足のない取引点です。
Q2【均衡計算】 需要関数 q = 400 − p、供給関数 q = p / 4 のとき、均衡 (p*, q*) は？
A(p*, q*) = (320, 80)
B(p*, q*) = (80, 320)
C(p*, q*) = (400, 0)
D(p*, q*) = (100, 300)
正解: A
D(p) = S(p) より 400 − p = p/4。両辺×4 で 1600 − 4p = p、移項して 1600 = 5p、よって p* = 320。q* = 320/4 = 80。
Q3【グラフ上の均衡】 横軸 q、縦軸 p のグラフ上で、Q2 の均衡点はどこに打つ？
A需要曲線と q 軸の交点
B横方向に 320、縦方向に 80 の位置
C原点 (0, 0)
D横方向に 80、縦方向に 320 の位置
正解: D
グラフは横軸 q、縦軸 p。均衡 (p*, q*) = (320, 80) は「横に q = 80、縦に p = 320」の点。順序の混乱を避けるには「横軸の値が q、縦軸の値が p」と毎回確認します。
Q4【超過需要・超過供給】 Q2 と同じ市場 (q = 400 − p, q = p/4) で、もし価格が p = 200 だったら何が起きる？
A取引がまったくなくなる
B需要量と供給量が一致して、ちょうど均衡になる
C需要量 200 > 供給量 50 となり、150 だけ超過需要（品不足）
D供給量 > 需要量 となり、超過供給（売れ残り）
正解: C
p = 200 のとき、需要 D(200) = 400 − 200 = 200、供給 S(200) = 200/4 = 50。需要 200 > 供給 50 で、150 だけ「超過需要」（品不足）が発生。一般に均衡価格より低いと超過需要、高いと超過供給が起きます。
Q5需要曲線が右にシフトすると、均衡価格 p* と均衡数量 q* はどうなる？
Ap* は下がり、q* は増える
Bp* は上がり、q* も増える
Cp* は下がり、q* も減る
Dp* は上がり、q* は減る
正解: B
需要右シフト = 同じ価格でも買いたい人が増える → 均衡価格も均衡数量も上がる。覚え方:「需要シフトは価格と数量が同じ向き」（右なら両方↑、左なら両方↓）。
Q6原材料費が高騰して、ある製品の供給が大きく減った。均衡 (p*, q*) はどう変わる？
Ap* も q* も上がる
Bp* は上がり、q* は減る
Cp* は下がり、q* も減る
Dp* は下がり、q* は増える
正解: B
原材料費高騰 → 供給左シフト → 同じ価格で作れる量が減る → 価格は上がり、数量は減る。覚え方:「供給シフトは価格と数量が逆向き」（右なら p↓ q↑、左なら p↑ q↓）。

  • 消費者余剰 (CS)・生産者余剰 (PS)・総余剰 (TS) の意味が分かる
  • 三角形の面積で CS・PS を計算できる

①確認クイズ（全 5 問正解で全 STEP クリア）
Q1消費者余剰 (CS) とはグラフでどの三角形の面積か？
A需要曲線の下、均衡価格の上にできる三角形
Bx 軸より下にできる三角形
C需要曲線と供給曲線の間にできる三角形
D均衡価格の下、供給曲線の上にできる三角形
正解: A
消費者余剰 (CS) = 需要曲線の下、均衡価格の上の三角形。「払ってもいいと思っていた額」と「実際に払った額」の差の合計です。
Q2家庭教師の市場 D(p) = 15000 − 3p、S(p) = 2p で均衡が (p*, q*) = (3000, 6000)。需要曲線の縦軸切片は 5000。CS の値は？
ACS = 600 万円
BCS = 900 万円
CCS = 300 万円
DCS = 1500 万円
正解: A
底辺 = q* = 6000、高さ = 需要切片 5000 − 均衡価格 3000 = 2000。CS = 1/2 × 6000 × 2000 = 6,000,000 = 600 万円。
Q3同じ家庭教師の市場で、PS（生産者余剰）の値は？（供給曲線の縦軸切片は 0）
APS = 900 万円
BPS = 180 万円
CPS = 1500 万円
DPS = 600 万円
正解: A
底辺 = q* = 6000、高さ = 均衡価格 3000 − 供給切片 0 = 3000。PS = 1/2 × 6000 × 3000 = 9,000,000 = 900 万円。
Q4同じ市場で、TS（総余剰）の値は？
ATS = 3000 万円
BTS = 600 万円
CTS = 1500 万円
DTS = 900 万円
正解: C
TS = CS + PS = 600 + 900 = 1500 万円。総余剰は市場が社会に生み出す「お得感の合計」で、均衡で最大になります。
Q5消費税を課したときの「死荷重 (DWL)」とは何か？
A消費者が払う総額のこと
B生産者の利益のこと
C取引が減ることで、誰の手にも渡らずに消える社会の損失
D政府が受け取る税収のこと
正解: C
消費税で取引量が減ると、本来成立していた取引から得られたはずの「お得感」が消えます。税収にもならず誰の手にも渡らない分が死荷重 (DWL) = 社会の損失です。
